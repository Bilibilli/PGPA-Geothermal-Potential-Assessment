# PGPA-Geothermal-Potential-Assessment
Python implementation of the Probabilistic Geothermal Potential Assessment (PGPA) framework.
The code is written in Python 3.9.23
First, create an environment based on Python 3.9.23. Then open the code file in the Visual Studio Code, and click "select Kernel" and choose the environment.
Second, According the tips in the beginning of the code, install the corresponding Python software libraries.
Third, change the information according to your own request, such as file path, lithology parameters and etc.
Then run the code and compile the results.
